### ğŸ§± HTML Structure Basics

```html

  <head> <!-- Metadata -->
  <body> <!-- Visible content -->
    <h3>Player Name</h3>
    <p>Salary Info</p>
  </body>
</html>
```

- `<h3>` â†’ heading  
- `<p>` â†’ paragraph  
- Tags can contain attributes like `class="info"`

---

### ğŸŒ³ HTML Document Tree

- HTML is structured like a tree  
- Tags = nodes  
- Children = nested tags  
- Siblings = tags inside the same parent

Example:
```html
<html>
  <head></head>   <!-- Child of <html> -->
  <body></body>   <!-- Sibling of <head> -->
</html>
```
