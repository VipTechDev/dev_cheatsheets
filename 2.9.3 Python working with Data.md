### ðŸ“‚ Reading & Writing Files

#### ðŸ”¹ File Opening Modes

| Mode | Purpose |
|------|---------|
| `'r'` | Read |
| `'w'` | Write (overwrite) |
| `'a'` | Append |
| `'r+'` | Read and write |
| `'b'` | Binary mode |

```python
with open("data.txt", "r") as file:
    content = file.read()

with open("output.txt", "w") as file:
    file.write("Hello, world!")

with open("log.txt", "a") as file:
    file.write("Log entry: Something happened.")

with open("data.txt", "r+") as file:
    content = file.read()
    file.write("Updated content: " + content)
```

## ðŸ“˜ Glossary: Working with Data in Python

This glossary defines key terms used in Python data workflows â€” from file handling to NumPy and Pandas. Use it to reinforce concepts and build cue-card rituals for deeper understanding.

---

| Term | Definition |
|------|------------|
| `.csv` file | A plain text file where each line is a row of data and values are separated by commas â€” great for storing tables. |
| `.txt` file | A basic text file that holds unformatted content â€” ideal for notes, logs, or raw data. |
| Append | To add something to the end of an existing file or list â€” like adding a new line to a log file. |
| Attribute | A property of an object in Python, accessed using dot notation (e.g., `df.shape`). |
| Broadcasting (NumPy) | A NumPy feature that lets arrays of different shapes work together in element-wise operations by stretching smaller arrays. |
| Component | An individual value inside a NumPy array â€” accessed using indexing like `array[0, 1]`. |
| Computation | Performing calculations on arrays or matrices â€” NumPy makes this fast and efficient. |
| Data analysis | The process of cleaning, exploring, and interpreting data to find insights and make decisions. |
| DataFrame | A Pandas table-like structure with rows and columns â€” perfect for structured data. |
| Dependencies | External libraries that a package relies on â€” Pandas depends on NumPy for core functionality. |
| File attribute | Metadata about a file, like its size or creation date â€” managed by the operating system. |
| File object | A Python object that represents an open file â€” used to read or write content. |
| Grid | A 2D layout of rows and columns â€” often used to represent tables or coordinate systems. |
| Hadamard Product | Element-by-element multiplication of two arrays with the same shape â€” result is a new array. |
| Importing Pandas | Use `import pandas as pd` to load the Pandas library and access its tools with the alias `pd`. |
| Index | A position used to access elements in a list, array, or DataFrame â€” starts at 0 in Python. |
| Libraries | Collections of pre-written Python code that add functionality â€” like NumPy for math or Pandas for data. |
| Linspace | A NumPy function that creates evenly spaced numbers between two values â€” useful for plotting or simulations. |
| NumPy | A Python library for fast numerical operations â€” supports arrays, matrices, and math functions. |
| One-dimensional NumPy | A NumPy array with a single row or column â€” like a list of numbers. |
| Open function | The built-in Python function `open()` lets you read from or write to files. |
| Pandas | A Python library for working with structured data â€” great for tables, time series, and analysis. |
| Pandas library | The full set of tools and modules within Pandas â€” includes DataFrames, Series, and data functions. |
| Plotting functions | Creating graphs or charts using libraries like Matplotlib â€” helps visualize data and trends. |
| Shape | The dimensions of a NumPy array â€” like `(3, 2)` for 3 rows and 2 columns. |
| Slicing | Selecting a portion of an array or list using index ranges â€” like `array[1:4]`. |
| Two-dimensional NumPy | A NumPy array with rows and columns â€” like a matrix or spreadsheet. |
| Universal Functions (ufuncs) | NumPy functions that apply to each element in an array â€” like `np.sqrt()` or `np.abs()`. |
| Vector addition | Adding two arrays element by element â€” result is a new array with summed values. |
| Visualizations | Graphs, charts, and plots that help you understand and present data â€” often made with Matplotlib or Seaborn. |
