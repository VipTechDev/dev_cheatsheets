### ðŸ“ˆ What Is a Series?

A Series is a one-dimensional labeled array.

```python
data = [10, 20, 30, 40, 50]
s = pd.Series(data)
print(s)
```

---

### ðŸ” Accessing Series Elements

```python
print(s[2])        # By label (value: 30)
print(s.iloc[3])   # By position (value: 40)
print(s[1:4])      # Range of elements
```

---

### ðŸ§° Series Attributes & Methods

- `values` â†’ Returns data as NumPy array  
- `index` â†’ Returns index labels  
- `shape`, `size` â†’ Dimensions and count  
- `mean()`, `sum()`, `min()`, `max()` â†’ Summary stats  
- `unique()`, `nunique()` â†’ Unique values  
- `sort_values()`, `sort_index()` â†’ Sorting  
- `isnull()`, `notnull()` â†’ Missing value checks  
- `apply()` â†’ Apply function to each element

---

### ðŸ§® What Is a DataFrame?

A DataFrame is a two-dimensional table with labeled rows and columns.

```python
data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'David'],
    'Age': [25, 30, 35, 28],
    'City': ['New York', 'San Francisco', 'Los Angeles', 'Chicago']
}
df = pd.DataFrame(data)
print(df)
```

---

### ðŸ“Š Accessing DataFrame Elements

#### ðŸ”¹ Columns
```python

print(df[['Name', 'Age']])  # Multiple columns
```

#### ðŸ”¸ Rows
```python
print(df.iloc[2])  # By position
print(df.loc[1])   # By label
```

#### ðŸ”¸ Slicing
```python
print(df[1:3])  # Rows 1 to 2
```

---

### ðŸ” Finding Unique Elements

### ðŸ”Ž Conditional Filtering

### ðŸ§° DataFrame Attributes & Methods

- `shape` â†’ Dimensions (rows, columns)

- `describe()` â†’ Stats for numeric columns  
- `head()`, `tail()` â†’ First/last rows  
- `mean()`, `sum()`, `min()`, `max()` â†’ Column stats

- `fillna()`, `drop()`, `rename()` â†’ Handle missing data, drop or rename columns  
- `apply()` â†’ Apply function to rows/columns
