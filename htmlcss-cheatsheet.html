CSS Overview & HTML5 Elements Cheatsheet

Element Description Example
<!DOCTYPE html> Defines document type <!DOCTYPE html><html>...</html>
<a> Anchor tag for hyperlinks <a href="https://www.ibm.com">IBM</a>
<article> Self-contained content block <article><h2>HTML</h2><p>...</p></article>
<aside> Content tangential to the main flow <aside>...</aside>
<body> Main content container <body>...</body>
<div> Generic container for styling <div>...</div>
<form> and <fieldset> Form structure and input grouping <form><fieldset>...</fieldset></form>
<figure> and <figcaption> Image container and description <figure><img src="..."><figcaption>...</figcaption></figure>
<header> Introductory section <header><h1>...</h1><p>...</p></header>
<nav> Navigation links <nav><a href="...">...</a></nav>
<section> Document sectioning <section><h2>...</h2><p>...</p></section>
<style> Internal CSS styles <style>p {color:red}</style>
<table> Data grid <table><tr><th>...</th></tr><tr><td>...</td></tr></table>
<ul>, <ol>, <li> Lists: unordered, ordered, and items <ul><li>...</li></ul>
<input> User input field <input type="text">
<img> Image embed <img src="path.jpg" width="300" height="300">
<script> JavaScript embed <script>alert("Hello World")</script>
<footer> Footer section of the document <footer>...</footer>
